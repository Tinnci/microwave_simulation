# 快速入门指南

## 简介

微波阻抗匹配设计工具是一个用于设计和优化微波阻抗匹配网络的软件。本指南将帮助您快速上手使用该工具。

## 安装

### 系统要求
- Python 3.12 或更高版本
- Rust 1.70 或更高版本
- Windows 10/11
- 2GB 以上内存
- 500MB 可用磁盘空间

### 安装步骤

1. 安装Rust工具链(如果未安装):
```bash
# Windows PowerShell
winget install Rustlang.Rust.MSVC
```

2. 克隆仓库：
```bash
git clone https://github.com/Tinnci/microwave_simulation.git
cd microwave_simulation
```

3. 安装Python依赖：
```bash
uv venv
uv pip install -r requirements.txt
```

4. 编译GUI程序：
```bash
cd gui_rust
cargo build --release
```

## 基本使用

### 启动程序

```bash
# 在项目根目录下
./gui_rust/target/release/microwave_gui
```

### 界面说明

1. 参数输入区
   - 工作频率（GHz）：输入范围1-20
   - 特征阻抗（Ω）：输入范围20-200
   - 负载阻抗：
     - 实部（Ω）：输入范围20-200
     - 虚部（Ω）：输入范围-200-200

2. 匹配方法选择
   - 四分之一波长变换器
   - 单枝节匹配器
   - L型匹配网络
   - π型匹配网络
   - T型匹配网络

3. 结果显示区
   - 匹配网络参数
   - S参数图
   - VSWR图
   - 优化状态

### 基本操作流程

1. 输入参数
   - 在相应的输入框中输入参数值
   - 输入框会实时验证数值的有效性
   - 无效输入会显示红色提示

2. 选择匹配方法
   - 点击对应的匹配方法按钮
   - 每种方法会显示其特有的参数设置

3. 计算结果
   - 点击"计算"按钮开始计算
   - 计算过程中会显示进度条
   - 结果会自动更新到显示区

4. 保存结果
   - 点击"导出"按钮
   - 选择要保存的数据类型
   - 选择保存位置

## 高级功能

### 参数优化

1. 优化设置
   - 选择优化目标(VSWR/反射系数)
   - 设置优化范围
   - 设置收敛条件

2. 运行优化
   - 点击"优化"按钮

1. 导出选项

2. 批量导出
   - 选择多个数据项
   - 设置导出格式
   - 选择保存目录

## 常见问题

### 输入验证错误

- 检查输入值是否在有效范围内
- 确保复数输入格式正确(a+bj)
- 验证频率单位是否正确(GHz)

### 计算错误

- 检查输入参数的物理合理性
- 确认选择的匹配方法是否适用
- 验证优化参数设置是否合理

### 程序启动问题

- 检查Rust环境配置
- 确认GUI程序编译成功
- 验证Python依赖安装完整
