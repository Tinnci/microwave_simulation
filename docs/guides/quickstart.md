# 快速入门指南

## 简介

微波阻抗匹配设计工具是一个用于设计和优化微波阻抗匹配网络的软件。本指南将帮助您快速上手使用该工具。

## 安装

### 系统要求
- Python 3.12 或更高版本
- Windows 10/11
- 2GB 以上内存
- 500MB 可用磁盘空间

### 安装步骤

1. 克隆仓库：
```bash
git clone https://github.com/Tinnci/microwave_simulation.git
cd microwave_simulation
```

2. 使用 uv 安装依赖：
```bash
uv venv
uv pip install -r requirements.txt
```

## 基本使用

### 启动程序

```bash
python src/gui/main.py
```

### 界面说明

1. 参数输入区
   - 工作频率（GHz）
   - 特征阻抗（Ω）
   - 负载阻抗（复数形式）

2. 匹配方法选择
   - 四分之一波长变换器
   - 单枝节匹配器
   - 双枝节匹配器（计划中）

3. 结果显示区
   - 史密斯圆图
   - S参数图
   - VSWR图
   - 优化结果

### 基本操作流程

1. 输入参数
   - 在工作频率框中输入频率值（1-20 GHz）
   - 在特征阻抗框中输入阻抗值（20-200 Ω）
   - 在负载阻抗框中输入复数阻抗值
     - 实部：20-200 Ω
     - 虚部：-200-200 Ω

2. 选择匹配方法
   - 点击相应的匹配方法单选按钮
   - 根据需要调整优化参数

3. 计算结果
   - 点击"计算"按钮
   - 等待计算完成
   - 查看结果显示区的图表和数据

4. 保存结果
   - 点击"保存"按钮
   - 选择保存路径
   - 选择保存格式（CSV/PNG）

## 高级功能

### 批量参数扫描

1. 进入批量模式
   - 点击"批量计算"按钮
   - 设置参数范围
   - 设置步长

2. 运行扫描
   - 点击"开始扫描"按钮
   - 等待计算完成
   - 查看扫描结果

### 优化计算

1. 设置优化目标
   - 选择优化参数
   - 设置目标值
   - 设置权重

2. 运行优化
   - 点击"优化"按钮
   - 等待优化完成
   - 查看优化结果

## 常见问题

### 输入验证错误

- 检查参数是否在有效范围内
- 确保复数输入格式正确
- 验证频率值是否合理

### 计算结果异常

- 检查输入参数的单位
- 确认匹配方法选择是否正确
- 验证优化参数设置

### 保存失败

- 检查保存路径权限
- 确保磁盘空间充足
- 验证文件名格式
