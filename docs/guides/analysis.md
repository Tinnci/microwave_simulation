# 微波阻抗匹配分析指南

## 结果分析

### S参数分析

#### S11（反射系数）
- 幅度表示反射损耗
- 相位表示反射波相位
- 理想值：0（完全匹配）
- 实际要求：<-10dB

#### S21（传输系数）
- 幅度表示插入损耗
- 相位表示传输相位
- 理想值：1（无损耗）
- 实际要求：>-3dB
